**aka Not sharing answers from class AT ALL**

## 😁👍 Overview

NSAFCISM (Non-Scholastic Academic File Collaboration and Integration System Manager) is a collaborative repository designed to streamline group project workflows for students. Rather than encouraging copying or shortcutting, this project focuses on providing structure and tooling that helps students develop their own solutions while collaborating effectively. It includes templates for organizing work, guidance on contribution etiquette, and utilities to help integrate separate contributions into a cohesive final product.

NSAFCISM serves as a central hub for organizing, versioning, and integrating these contributions. The repository provides clear conventions for file naming and submission, simple automation to run tests and checks, and documentation to help teams collaborate without losing individual accountability. Our goal is to make it easier for student teams to produce polished, well-rounded final deliverables while maintaining academic integrity.

## Getting Started

To contribute to this project, follow these steps:

1. Clone the repository: `git clone https://github.com/lowercasemonty/nsafcism.git`
2. Navigate to the project directory: `cd nsafcism`
3. Install necessary dependencies (if any)
4. Create your own solutions to all problems following the naming format of "NOTcheatingat-(project_name)-(Your Name)"
